# Blood Bank Management System

A web-based Blood Bank Management System built with PHP, MySQL, HTML, and CSS that helps manage donors, recipients, and blood inventory efficiently.

## Project Overview
This project allows users to register as blood donors or recipients, enables admins to manage donor records and blood stocks, and provides real-time information on blood availability. It streamlines the process of finding and requesting blood units, improving coordination in critical situations.

## Features
- Donor registration and login system  
- Recipient blood request functionality  
- Admin dashboard to add, view, update, and delete donor records  
- Real-time blood stock tracking based on blood group  
- User-friendly interface with responsive design


## Technology Stack
- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** PHP  
- **Database:** MySQL  
- **Environment:** XAMPP / WAMP

## Installation & Setup
1. Clone the repository:  
```bash
git clone https://github.com/Jvssankar/Blood-bank.git
```
2. Move the project folder to the web server directory:
  htdocs for XAMPP
  www for WAMP
3. Start Apache and MySQL using the control panel.
4. Create a database named blood_bank in phpMyAdmin.
5. Import the SQL file (if provided) to populate tables.
6. Open the browser and visit:
   http://localhost/Blood-bank

## How to Use
1. Register as a donor or request blood as a recipient.
2. Admin can log in to manage donors and monitor blood stock.
3. View available blood units and process requests quickly.
